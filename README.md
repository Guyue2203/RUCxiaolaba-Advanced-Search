# RUCer苦小喇叭久矣！
本项目是个个人开发的草台班子项目，旨在改善用户的小喇叭搜索体验。


校园网环境下用  http://10.46.140.197:8080   即可访问



网站原理为每天凌晨三点到四点停机爬取小喇叭前一天的内容，其他时间可对已爬内容进行搜索。



相比小喇叭原本的搜索有以下优势：
* 可多关键词搜索
* 可以搜索到评论内容
* 数据保存时间长（小喇叭会删除一年以上的数据）
* 不受某些删帖影响（如果爬的时候没删就不影响）
* 可以用AI总结
